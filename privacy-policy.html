Welcome

Welcome to Link Safe: a powerful, AI-driven moderation solution built for Discord servers. This Privacy Policy outlines how Link Safe (the “Service”) handles your data and what you can expect when using it.

Introduction

Link Safe is developed and maintained by @linksafe (the “Developer”) with the goal of protecting Discord communities from malicious activity. By using the Service, you agree to the terms described in this Privacy Policy.

Data Collection & Storage

  • The Service may store the following data:
    • Message Content:
      For logging purposes, when a message is edited or deleted, a temporary OneTimeSecret link is generated with automatic expiry automatically set to 30 days of inactivity, or after the link has been accessed.
      • This link is protected with a randomly-generated password (32 characters long).
      • This link is stored securely within OneTimeSecret servers in the European Union.

    • User ID:
      An identifier that never changes and is associated with a Discord account.
      • Two-Factor Authentication (2FA) system for administrative commands.

  • *Sudo Mode: A SHA3-512 (Keccak Sponge Construction) one-way, non-reversible hash with ~700,000 iterations is computed from the passphrase you provide.
  • No private information is retained passively or without your knowledge.
